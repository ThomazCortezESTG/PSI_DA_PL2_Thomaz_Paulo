# Projeto IShopping

Este projeto tem como objetivo ajudar utilizadores a fazer compras uma lista de compras com base num orçamento definido pelo utilizador, realizar a compra e no fim imprimir um recibo caso  outilizador o desejar.
---

## Funcionalidades

- Definir um orçamento máximo por mês
- Fazer uma lista de compras
- Adicionar produtos ao carrinho
- Calcular o total das compras
- Verificar se o orçamento foi ultrapassado
- Gerar recibo
- Ver o histórico de compras
- Calcular estatisticas

---

## Tecnologias Utilizadas

- Linguagem: C#
- Interface: Windows forms
